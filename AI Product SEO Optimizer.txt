# Shopify App Project: AI Product SEO Optimizer

I want to build a Shopify embedded app using Shopify Remix, React, Polaris, PostgreSQL, and Node.js.

## Goal

Create an AI-powered Shopify app that helps merchants optimize:

* Product Title
* SEO Title (Meta Title)
* Meta Description

The app should use AI providers such as:

* OpenAI
* Claude
* Gemini

Merchants can choose which provider to use.

---

# Main Problem We Solve

Many Shopify stores have:

* Poor product titles
* Duplicate titles
* Generic imported product names
* Weak SEO metadata

Our app should improve SEO and click-through rates using AI.

---

## 1. AI Product Title Generator

"Bluetooth Wireless Earbuds with Noise Cancellation"

## 4. Bulk Processing

## 5. Version History System

IMPORTANT:

Never overwrite data permanently.

Create version history.

Database table example:

title_versions

* id
* shop_id
* product_id
* content_type
* old_value
* new_value
* created_at

Supported content types:

* Product Title
* SEO Title
* Meta Description

Merchant can restore any previous version.

Store versions indefinitely.
